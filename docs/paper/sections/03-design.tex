\section{Design}

This section is intentionally a routing stub. The stable project invariant is
that \namei is a narrow VFS name-resolution extension point, not a BPF
filesystem, access-control framework, mount namespace replacement, or custom
configuration language.

Current mechanism scope and non-goals live in \file{docs/idea-story.md}.
Implementation evidence must be recorded in dated files under
\file{docs/tmp/YYYY-MM-DD-*.md} with raw artifacts under \file{results/}.
