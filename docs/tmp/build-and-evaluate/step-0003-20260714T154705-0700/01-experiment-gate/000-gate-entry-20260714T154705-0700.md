frozen claim on the strongest source-backed agent filesystem/workspace setting.
A positive result can show that `namei_ext` satisfies a real agent workspace
path-view oracle through the real KVM attach path while lower filesystem
semantics remain owned below the hook. A contradictory result would identify
whether the source oracle actually requires synthetic contents, custom metadata
persistence, write-conflict handling, data-path mediation, or other broader
filesystem ownership.

supporting implementation artifacts, but prior review classified the current
matrix as too synthetic and incomplete for Experiment A. The next work must
therefore repair or admit the complete source-derived matrix rather than add
another weak baseline or smoke test.
